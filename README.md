# Tamar Khatiashvili — Portfolio

შავი მინიმალისტური პორტფოლიო ოქროსფერი აქცენტებით, ხმოვანი ეფექტებითა და ადმინ პანელით.

## სწრაფი სტარტი

1. გახსენი `index.html` ბრაუზერში — ეს არის მთავარი პორტფოლიო საიტი.
2. ადმინ პანელისთვის გახსენი `admin.html` ან დააკლიკე ფუტერში "Admin".
3. **ნაგულისხმევი პაროლი:** `tamar2026` (Settings ტაბში შეგიძლია შეცვლა).

## ფაილური სტრუქტურა

```
portfolio/
├── index.html          ← პორტფოლიო საიტი
├── admin.html          ← ადმინ დეშბორდი
├── README.md
└── assets/
    ├── css/style.css   ← შავი/ლუჯი + ოქრო თემა
    ├── js/
    │   ├── data.js     ← ნაგულისხმევი კონტენტი + localStorage
    │   ├── main.js     ← საიტის ლოგიკა, ხმები, ანიმაციები
    │   └── admin.js    ← ადმინის CRUD ლოგიკა
    ├── img/profile.jpg ← შენი ფოტო
    └── sounds/         ← (ხმები გენერირდება Web Audio API-ით)
```

## ადმინ პანელი

**6 ტაბი:**

- 📊 **Overview** — სტატისტიკა, JSON ექსპორტი/იმპორტი
- 👤 **პროფილი** — ტექსტები hero-სა და about სექციებში
- 🚀 **პროექტები** — დაამატე/რედაქტირება/წაშლა (GitHub URL, აღწერა, სურათი, stack)
- 💼 **გამოცდილება** — კარიერული timeline (პერიოდი, როლი, ბულეტები, რიგის შეცვლა)
- ⚡ **სქილები** — ცალ-ცალკე დამატება ან ბევრი ერთად
- ⚙ **პარამეტრები** — პაროლის შეცვლა, ნაგულისხმევზე დაბრუნება

## დეპლოი

ვინაიდან ეს არის სტატიკური საიტი, შეგიძლია უფასოდ განათავსო:

- **GitHub Pages** — push რეპოში, settings → pages → enable
- **Netlify** — drag & drop ფოლდერი https://app.netlify.com/drop
- **Vercel** — `vercel deploy` ბრძანებით ან გითის integration-ით
- **Cloudflare Pages** — დაუკავშირე GitHub რეპო

## შენიშვნა მონაცემების შესახებ

ყველა ცვლილება ინახება ბრაუზერის `localStorage`-ში. რომ კონტენტი
გადაიტანო სხვა ბრაუზერზე ან კომპიუტერზე:

1. Overview ტაბში დააკლიკე **JSON ექსპორტი** — ჩამოიწერება ფაილი.
2. სხვა ბრაუზერში გახსენი ადმინი → **JSON იმპორტი** → ატვირთე ფაილი.

თუ გინდა რომ კონტენტი ერთიანი იყოს ყველა ვიზიტორისთვის (და არა
მხოლოდ შენთვის), შემდეგ ეტაპად შეგვიძლია backend-ი (Node + DB)
ჩავუმატოთ.
